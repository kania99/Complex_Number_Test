## Setup
Prerequisites:
* Installed CMake *(minimum VERSION 3.0)*,

Clone repository:

    git clone https://github.com/kania99/Complex_Number_Test.git

Create `build` directory and launch `CMake`:

    cd Complex_Number_test
    mkdir build
    cd build
    cmake ..

Run `make` command and execute the program:

    make
    ./Complex_Number_Test
